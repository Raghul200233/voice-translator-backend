## Project Setup
This project converts audio to text using OpenAI Whisper API.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Tailwind CSS
- OpenAI Whisper API

## Setup Instructions
1. Clone the repository
2. Run \`npm install\` in both frontend and backend directories
3. Create \`.env\` file with API keys
4. Run \`npm run dev\`" > README.md
